Hackathon “Hack 4 Smart Cycling”

Took first place in a Hackathon , where I developed a prototype for a smart cycling system with my team member Leon Staubach.

Idea: Use Machine Learning and available data to create round trip routes for cyclists, based on POI categories that they are interested in. We used DBSCAN to create clusters of similar POIs, build routes between them and reduce the length of each round trip route to the maximum length that a user had set.
